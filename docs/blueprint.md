# **App Name**: NeuraFolio

## Core Features:

- Hero Section Profile Display: Showcase the AI Engineer's name, role with a dynamic typing animation, and an AI-styled avatar/hologram in a highly interactive and futuristic hero section.
- Dynamic Skill Visualization: Interactive skill cards covering AI/ML, Backend, Cloud, Data Engineering, and Frontend domains, featuring icons and animated progress bars to demonstrate proficiency.
- Interactive Project Showcase: Display featured projects through detailed, interactive cards with animated hover effects, technology stack badges, and direct links to GitHub repositories and live demos. Projects are filterable by AI categories.
- AI Playground Interaction Tool: A unique interactive section where visitors can engage with a simplified generative AI model via text input, serving as a practical tool to demonstrate the AI engineer's expertise in applied AI.
- Responsive Layout & Modern Components: Ensures the application is fully responsive and optimized for both desktop and mobile devices, integrating modern UI components like cards, spotlight hover effects, animated gradients, glowing borders, and glassmorphism elements.
- Advanced Motion & Micro-interactions: Implements smooth scrolling, elegant loading animations, scroll-triggered reveal animations, and subtle cursor glow effects using Framer Motion to create an engaging and professional user experience.
- Dynamic 3D Background & Visuals: Integrates an interactive particle network animation as a background element using Three.js or React Three Fiber, complemented by an AI pattern grid and floating gradient orb, reinforcing the futuristic AI theme.

## Style Guidelines:

- Overall Color Scheme: Dark mode is default, conveying a sophisticated, high-tech, and futuristic ambiance across the entire application.
- Primary Color: A vibrant electric blue (#3388FF), chosen for its strong association with technology, innovation, and its striking contrast against a dark background, evoking a cutting-edge aesthetic.
- Background Color: A deep, almost black, but subtly bluish tone (#111215), providing a sleek, professional canvas that allows the primary and accent colors to stand out prominently, consistent with a dark theme.
- Accent Color: A bright neon purple (#D255F9), used for secondary highlights, interactive elements, and to introduce a vibrant cyberpunk-minimalist energy, complementing the electric blue effectively.
- Headline font: 'Space Grotesk' (sans-serif), chosen for its technical, computerized, and modern appearance, ideal for titles and prominent textual elements that need to convey a futuristic feel.
- Body font: 'Inter' (sans-serif), paired with Space Grotesk for body text, ensuring excellent legibility and maintaining a clean, objective, and professional aesthetic for longer content sections such as the 'About Me' and 'Experience' areas.
- Use sharp, vector-based icons that align with a minimalist tech aesthetic, emphasizing clear lines and modern geometric shapes. Icons for skills and social links should feature subtle glowing effects upon hover to enhance interactivity.
- Employ a responsive grid-based layout with ample negative space to achieve a clean, modern, and premium feel, utilizing glassmorphism for layered UI elements like testimonial cards and project previews to create depth.
- Implement smooth page transitions, subtle micro-interactions for buttons and interactive cards (e.g., spotlight hover, glowing borders), elegant loading animations, and dynamic reveal animations triggered on scroll to provide a highly engaging and fluid user experience.
